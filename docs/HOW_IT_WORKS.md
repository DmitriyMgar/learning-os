# Как работает Learning OS

Эта система — это обычный Python-проект с профессиональной структурой.

## Основные принципы

1. Любое задание = Python-функция
2. Любое задание проверяется тестом
3. Успех = make check без ошибок
4. Всё хранится в Git
5. Среда воспроизводима через Codespaces

## Команды

make install   — установка зависимостей
make lint      — проверка стиля
make test      — запуск тестов
make format    — автоформатирование
make check     — lint + test
make progress  — обновление прогресса

## Импорты

Папки с кодом всегда являются Python-пакетами:
- содержат __init__.py
- корень проекта добавлен в PYTHONPATH через pytest.ini

## Среда

Среда описана в .devcontainer/devcontainer.json.
При пересборке Codespace всё устанавливается автоматически.
