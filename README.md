# Learning OS — моя обучающая система Python

Это мой личный облачный учебный репозиторий.

Цели:
- Учиться Python и backend системно
- Работать с любого устройства через GitHub Codespaces
- Хранить код, задания, тесты и прогресс в одном месте
- Учиться как разработчик, а не как читатель

Основная команда проверки:

make check

Если она проходит — код и задание считаются выполненными.

Структура:
- syllabus/  — программы курсов
- lessons/   — теоретические уроки
- kata/      — мои решения
- tests/     — автопроверка
- progress/  — мой прогресс
- scripts/   — вспомогательные утилиты
